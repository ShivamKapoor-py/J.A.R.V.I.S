# 🤖 J.A.R.V.I.S (Just A Rather Very Intelligent System)

## 🚀 About Project
J.A.R.V.I.S is a web-based AI assistant that can perform tasks using voice commands, search queries, and intelligent responses.

## ✨ Features
- 🎙 Voice Recognition (Speech-to-Text)
- 🔊 Text-to-Speech
- 🤖 AI Chatbot (Google Search API)
- 🔍 Smart Search System
- 🔐 Login System
- 💻 Modern UI with animations

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- Web Speech API
- Google Custom Search API

## ▶️ How to Run
1. Download or clone the repo
2. Open `index.html` in browser
3. Use microphone button for voice commands
